# Генератор случайных аватарок

Сайт позволяет сгенерировать случайную аватарку для вашего профиля.

## Использование

Нажмите на кнопку "Сгенерировать аватарку". Изображение можно скачать через контекстное меню.

## Особенности реализации

- Контроль состояний через `useState`;
- Нет асинхронности, т.е. все происходит мнгновенно;
